Install and Configurations

No complicated xml configuration files. Just download the latest software, setup your database and basepath to start your awesome application.

Creating Controller

To get started, lets create our first controller inside the apps/controllers directory. Example- 


    namespace Apps\Controllers;

    use Cygnite;
    use Cygnite\Cygnite as Cf;
    use Cygnite\Loader\CF_BaseController as CF_BaseController;
    use Cygnite\Helpers\GHelper as GHelper;

    class Home extends CF_BaseController
    {
            public function __construct()
            {
                parent::__construct();
                Cf::loader()->session->save('author','Sanjay from Session');
            }

            public function action_index()
            {
                echo 'Hello World!!';
            }
    }

Creating Model

Lets create our first model inside the apps/models directory. Example- 

    namespace Apps\Models;

    use Cygnite\Cygnite as cf;
    use Cygnite\Sparker\CF_BaseModel as CF_BaseModel;
    Cf::import('packages.cygnite.sparker.cf_basemodel');

    class Activerecords extends CF_BaseModel
    {

        public function __construct()
        {
            parent::__construct('cygnite');//Your database name
        }
    }

Creating View

Next, lets create simple view to display our welcome page which is reside in apps/views directory. Example- 

<html>
    <body>
        <h1>Cygnite Framework Quickstart</h1>
            Congrats !! You have created your first view page.
    </body>
</html>

You can also create template layout and render the sections into it. For more details check Views.

Routing

Cygnite Framework has powerful routing. You can use routerconfig.php and router.php inside apps/ directory for routing. Let us create our first routing. Example -

1. routerconfig.php for direct routing. 

return array(
    '/sayhello/(\w+)/(\d+)'             => 'home.test',
    '/blog(/\d{4}(/\d{2}(/\d{2}(/[a-z0-9_-]+)?)?)?)?'          => 'home.runcrons'
);
1. router.php has closure support. 

   $router = Cygnite::loader()->router;
    $GLOBALS['router'] = $router;

    // Before Router Middleware
    $router->before('GET', '/.*', function() {
          echo "Cygnite Framework has awesome features ";
          exit;
    });
    $router->run();

For brief details please have a look into routing guide.

Display data into view

Let us see how to display data into view. 

 $this->render("welcome")->with(array(
                                     'author'=>$this->author,
                                    'email'=>'sanjoy09@hotmail.com',
                                    'messege' => 'Welcome to Cygnite Framework',
                                   ));
You can get the data into the view or in layout as follows 
    echo $this->params['author'];
     echo $this->params['email'];
