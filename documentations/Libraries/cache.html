Cache

Cygnite Framework allows you to cache your page in three popular ways-

  • i. File Cache
  • ii. APC Cache
  • iii. Memcache

Configurations

You must configure in apps/configs/config.php in order to use cygnite file cache system.

  • i. Enable cache in your config file by TRUE/ FALSE
  • ii. Give your cache name which name cache should generate. Default value is cf_cache.
  • iii. You can also change your cache file extensions by config.
  • iv. Provide cache type equal to filecache to generate your file cache.
  • v. Your cache files are stored into your apps/temp/cache/ folder. You can optionality change your path in config.

How to cache your page ?

Caching page is very simple via your controller- For example-


   Cygnite::loader()->filecache->write_cache('welcomepage', $this->render("welcome",TRUE));
In above example the render method will return your view page as string which used to store in cache.

How to retrieve your cached page ?

Your cached page are stored into the file as json format which can be retrieve via your cache key. You can pass third parameter expire time to destroy your cached page. 

     echo Cygnite::loader()->filecache->read_cache('welcomepage');

APC Caching

Store into APC

You can store your data or page into APC cache to boost up your application performance. For example- 

   use Cygnite\Cygnite;

    Cygnite::loader()->apc->store($key,$value);

Set lifetime of APC cache

You can set your cache life time by - 

   use Cygnite\Cygnite;

    Cygnite::loader()->apc->set_life_time('Your time in sec');
If you don't set life time Cygnite will use default life time as 600s.

Retrieve your data from APC

Fetch data from APC - 

   use Cygnite\Cygnite;

    Cygnite::loader()->apc->fetch('Your cache key');

Manually destroy your data from APC

You can also destroy the data from APC by the key- 

   use Cygnite\Cygnite;

    Cygnite::loader()->apc->destroy('Your-cache-key');

Memcache

You need have memcache configured to use this library.

Add your server

Call add server method to initialise the server. By default cygnite connects with localhost and port number 11211. 

   use Cygnite\Cygnite;

    Cygnite::loader()->cfmemcache->add_server($host,$port);

Store data into memcache

Store your data into memcache by key- 

   use Cygnite\Cygnite;

    Cygnite::loader()->cfmemcache->store($key,$value);

Fetch data from memcache

   use Cygnite\Cygnite;

    Cygnite::loader()->cfmemcache->fetch($key);

Destroy your data from memcache

   use Cygnite\Cygnite;

    Cygnite::loader()->cfmemcache->destroy($key);
That's all about the caching system. We are constantly upgrading core files to build it better.
