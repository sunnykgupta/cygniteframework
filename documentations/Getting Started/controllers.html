Controllers

How to set default Controllers ?

You can set your default controller of your application simply changing configuration in apps/configs/config.php file. For example- 

     return array(
               'default_controller'    => 'home',
               'default_method'          => 'index',
    );
You can also change the default action of your controller in above configuration.

Controller

Controllers are resides in apps/controllers/ directory.

  1. All controllers must have nameapces included in the top of the page.
  2. Controllers should extends CF_BaseController.
  3. Controllers should have constructor and it should call parent constructor.
  4. Controllers all methods should have prefixed with action_method_name.
  5. You can import classes by use keyword under namespaces.
For example let us create a simple controller to render view and see how it is works- 
 namespace Apps\Controllers;

    use Cygnite\Cygnite;
    use Cygnite\Loader\CF_BaseController;
    use Cygnite\Helpers\GHelper;

    class Home extends CF_BaseController
    {
        public function __construct()
        {
             parent::__construct();
        }

        public function action_index()
        {
          $this->render('welcome',array( 'title'=> "Welcome to Cygnite Framework"));
        }
    }
You must create your view page inside apps/views/welcome.view.php page to render it.

We are done ! Now lets call index action via browser url to view output. For example- 

http://www.example.com/cygnite/index.php/home/index
