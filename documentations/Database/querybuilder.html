Database Active Records

How to select from Database ?

Now selecting data from database is very simple then you think. You can select your data from table in a single line of code. For simple select example - 

    $data= $this->select('all')->fetch_all('registration');
    $this->flushresult();

Where Condition

Select data with where condition. For simple select example - 

   $this->where('name','Sanjay','=');
Write your multiple where conditions with AND - 
    $where = array(
            'name =' => 'Sanjay',
            'id >'=> '4'
        );
  $this->where($where);

The above query will generate multiple where conditions as below-

Select `username`,`password` from `users` where `username` = 'Sanjoy' AND `id ` > 4;

[Note: Limitation - Don't pass a single conditions into array format. It will give unknown result.]

Order By

Write order by as below - 

    $this->order_by('id','DESC');

Group By

Write your group by condition to fetch data. For example - 

    $this->group_by(array('name'));

    For multiple fields

    $this->group_by(array('name','id'));

Limit Queries

Limit your queries as below. 

   $this->limit(3);

   LIMIT 0,3

   $this->limit(1,3);

All together in a single line

You can now execute all together in single line as method chaining. For example- 

  $this->where('name','Sanjay','=')
         ->group_by(array('name'))
         ->order_by('id','DESC')
         ->limit(3)
         ->fetch_all('users');

Insert Query

Cygnite Framework build your queries active records style. You can insert your data into db as below- 

     $this->name = "Cygnite Active Records !!";
     $this->entry_date = date('Y-m-d H:m:s');
     $this->comment = 'Insert query http://www.cygniteframework.com';
     $this->save('users');

Every single properties act as your table fields and finally save method to save your data into table. Its very simple insert data now.

Update Query

Update also similar as insert. You can update your row as below- 

    $this->name = "Cygnite Active Records Update Query";
    $this->entry_date = date('Y-m-d H:m:s');
    $this->comment = 'Update query http://www.cygniteframework.com';
    $this->save('user',array('id'=> 25));

Here also write your properties to update your table fields and finally save method to save your data into table.

Delete Query

Delete rows based on your column key. For example- 

    $this->id = 26;
    $this->remove('users');
